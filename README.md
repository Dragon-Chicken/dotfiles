# Dependencies
- eww
- nvim
- picom
- polybar (going to be replaced by eww)
- SXWM
- zsh

# Installation
1. Download the dependencies
1. Clone this repo
```sh
git clone https://github.com/Dragon-Chicken/dotfiles.git
```
3. Install
```sh
make # or `make install`
```
4. Uninstall
```sh
make uninstall
```
